# 道学文化与命理测算工具集

这是一个集合了道教文化、命理测算和数字游戏的传统文化娱乐工具网站。

## 项目介绍

本项目包含多个独立的传统文化主题工具页面，旨在传承和推广中华传统文化，提供娱乐性的命理测算和道教文化学习工具。

## 工具列表

1. **道教八大神咒页面** (`zhou.html`) - 学习道教经典八大神咒，了解其含义与用途
2. **生辰八字查询工具** (`bazi.html`) - 输入阳历生日时间，查询对应的阴历生辰八字
3. **手机号81数理解读工具** (`81.html`) - 通过手机号码末四位数字，解读对应的81数理吉凶
4. **索引页面** (`index.html`) - 主页面，提供所有工具的导航入口

## 使用说明

直接在浏览器中打开相应的HTML文件即可使用各工具。

- 纯静态HTML实现，无需服务器支持
